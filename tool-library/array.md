# 阵列

可以在 FormIt 中创建线性阵列和径向阵列。要**阵列边、面或整个对象**，请单击上下文菜单中的“阵列”图标。接下来，选择“线性”或“径向”、指定其他参数，然后单击对象以开始创建阵列。再次单击以放置阵列的图元。
