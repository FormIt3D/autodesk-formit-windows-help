# 特性

在 Formit 工作平面中，**“特性”选项板**会显示与选定对象相关的信息和数据。

使用“特性”选项板，可以编辑对象编组、将对象指定给图层以及更改对象的材质。

![](../.gitbook/assets/properties\_palette.png)
