# 性質

**性質選項板**顯示與 Formit 工作平面中所選物件相關的資訊和資料。

使用「性質」選項板，您可以編輯物件群組、將物件指定到圖層，以及變更物件的材料。

![](../.gitbook/assets/properties\_palette.png)
