# 模型统计信息

“模型统计信息”提供了一种查看 FormIt 模型统计信息的方法。

通过转到“窗口”菜单并选中“模型统计信息”旁边的框，可启用“模型统计信息”。

![](../.gitbook/assets/ModelStatisticsMenu.png)

在“模型统计信息”中，将显示模型级别统计信息，然后显示每个组的统计信息（标识为历史记录）。此处显示了模型级别统计信息的示例。
