# モデルの統計

[モデルの統計]では、FormIt モデルの統計を表示できます。

[モデルの統計]を有効にするには、[ウィンドウ]メニューに移動して[モデルの統計]のチェックボックスをオンにします。

![](../.gitbook/assets/ModelStatisticsMenu.png)

[モデルの統計]にはモデル　レベルの統計情報が表示され、その後に各グループの統計情報が履歴として提示されます。モデル レベルの統計情報の例を次に示します。
