# FormIt + Revit

![](<../.gitbook/assets/2017 formit + revit hero image.jpg>)

## FormIt 的易用性符合 Revit 的強大功能 <a href="#revit-intro" id="revit-intro"></a>

FormIt for Windows 直接連接至 Revit，方便在早期概念與詳細設計之間順暢地溝通。

## FormIt + Revit 的新增功能 <a href="#revit-what-s-new" id="revit-what-s-new"></a>

### **FormIt 2023.1 改進功能** <a href="#formit-20221-improvements" id="formit-20221-improvements"></a>

* 透過「3D 草圖」按鈕啟動時，在 Revit 中設定的位置現在會同步到 FormIt。
* 改善透過「3D 草圖」按鈕啟動 FormIt 時，DWG 線條處理的同步。
* 在「3D 草圖」中或編輯連結的 FormIt 物件時，可選擇在 FormIt 和 Revit 之間同步相機。

### 2023 的連結威力 <a href="#formit-20221-improvements" id="formit-20221-improvements"></a>

* 使用 Revit「插入」頁籤中的 _**「連結 CAD」**_ 按鈕，可將 FormIt 模型 (AXM 格式) 連結至 Revit。
* 在 Revit 中選取連結的 FormIt 元素，然後使用新的 _**「以 FormIt 編輯」**_ 按鈕，即可在 FormIt 中啟動該元素立即進行編輯。
* 使用「圖層」選項板中新的 _**「傳送至 Revit」欄**_，可控制在 FormIt 草圖連結至 Revit 時，要將哪些圖層傳送至 Revit。
* 使用「圖層」面板底部，「傳送至 Revit」按鈕旁邊新的 _**「預覽傳送至 Revit 的選取項目」**_ 模式，可預覽將傳送至 Revit 的幾何圖形。
* 現在會為 Revit 中的 FormIt 物件繪製剪影邊緣。

### **FormIt 2022.1 改進功能** <a href="#formit-20221-improvements" id="formit-20221-improvements"></a>

使用「3D 草圖」按鈕從 Revit 啟動 FormIt，並在 FormIt「樓層」選項板中包括所有或選取的 Revit 樓層資料。

### **2022 版全新功能** <a href="#all-new-for-2022" id="all-new-for-2022"></a>

* 使用 Revit「量體與敷地」頁籤上新的 _**「3D 草圖」**_ 按鈕，可直接從 Revit 啟動 FormIt。
* 如果從 _「3D 草圖」_ 按鈕啟動，使用 FormIt 中的 _**「傳送至 Revit」**_ 按鈕，可在 FormIt 中進行設計並將草圖傳回 Revit。
* 使用 Revit「插入」頁籤中的 _**「匯入 CAD」**_ 按鈕，可將 FormIt 模型匯入 Revit (比先前的 _「將 FormIt 匯入至 RVT」_ 按鈕更可靠且最多**快 30 倍**)。
* 無需下載單獨的增益集，就能取得 FormIt 功能。

### **Revit 2021.1 改進功能** <a href="#revit-20211-improvements" id="revit-20211-improvements"></a>

* 改善邊可見性的一致性：FormIt 中的平滑/隱藏邊在 Revit 中仍會隱藏。
* 改善材料方向的一致性：在 FormIt 面上具有自訂比例/旋轉取代的材料在 Revit 中會保留這些取代。
* 修正從 Revit 轉換 FormIt 幾何圖形會失敗或導致 Revit 當機的幾種情況。

## 開始使用 FormIt + Revit <a href="#revit-workflows" id="revit-workflows"></a>

### **直接從 Revit 以直覺方式繪製 3D 草圖** <a href="#intuitive-3d-sketching-directly-from-revit" id="intuitive-3d-sketching-directly-from-revit"></a>

如果您同時處理早期階段和詳細設計，可以在 FormIt 和 Revit 之間快速重複並順暢地探索設計。

* 只需按一下，即可將關聯式的幾何圖形從 Revit 推送至 FormIt。
* 在 FormIt 中設計和塑型，並將您的設計在同一個工作階段傳回同一個 Revit 專案。
* 在 Revit 中使用 _「連結 CAD」_ 工具參考 FormIt 模型，並繼續在 Revit 中進行視覺化和詳圖製作。
* 從 Revit 使用 _「以 FormIt 編輯」_ 啟動 FormIt 連結。

![](<../.gitbook/assets/revit to formit.png>)

### **3D 草圖、以 FormIt 編輯，以及傳送至 Revit 以進行設計重複過程** <a href="#3d-sketch-and-send-to-revit-for-design-iteration" id="3d-sketch-and-send-to-revit-for-design-iteration"></a>

### **建議的工作流程**

* 使用「剖面框」定義 Revit 模型的內部部分，然後按一下 _**「3D 草圖」**_ 按鈕開啟 FormIt，探索內部配置或預定的設計功能。按一下將新設計傳送至 Revit。
* 從已在 Revit 中塑型的城市環境開始，然後按一下 _**「3D 草圖」**_ 按鈕，將周圍環境帶入 FormIt 做為可鎖點的參考。按一下將新設計傳送至 Revit。
* 開啟既有的 Revit 建築模型，然後使用 _**「3D 草圖」**_ 按鈕啟動 FormIt。為新的增建部分的幾個設計塑型，將每個設計放在不同的 FormIt 圖層上。
* 在 Revit 中選取 FormIt 物件，然後使用 _**「以 FormIt 編輯」**_ 按鈕，在 FormIt 中啟動模型。

![](../.gitbook/assets/edit-in-formit.png)

![選取 FormIt 物件後使用「以 FormIt 編輯」](../.gitbook/assets/revit-edit-in-formit-workflow.gif)

* 使用 _**Revit 預覽模式**_ 預覽將傳送至 Revit 的幾何圖形，以確保您共用的是要用於詳細開發的正確設計。
* 從設計中選取您想要的元素，然後使用 _**「傳送至 Revit」**_ 按鈕或儲存連結模型，將每個設計傳送至 Revit。

### **使用工具**

* _**「3D 草圖」**_ 按鈕、 _**「以 FormIt 編輯」**_ 按鈕和 _**「傳送至 Revit」**_ 工作流程可供 Autodesk 工程建設 (AEC) 軟體集的訂購客戶和具有特定存取權類型的客戶使用。透過 Revit LT 無法使用。[從這裡進一步瞭解工程建設軟體集](https://www.autodesk.com/collections/architecture-engineering-construction/overview)。

### 「3D 草圖」按鈕

![使用 3D 草圖啟動新例證，然後使用「傳送至 FormIt」儲存並建立 Revit 連結](../.gitbook/assets/revit-3d-sketch-send-to-revit.png)

* 當您啟動 Revit 2022.0 或更高版本時，「量體與敷地」頁籤上會有一個稱為 **「3D 草圖」** 的新按鈕。此按鈕只能從 3D 視圖使用；在 Revit 的 2D 視圖中將顯示為停用。
* 請確保您已使用自己的 Autodesk 認證登入 Revit。啟動 _**「3D 草圖」**_ 按鈕，可能會提示您再次登入 Autodesk 帳戶。這是為了檢查您是否擁有正確的權利。
* 如果未安裝 FormIt 2022.0 或更高版本 (或安裝了 FormIt 2022.0 以前的版本)，則不會啟動 FormIt，而會提示您下載正確的 FormIt 版本。

#### 3D 草圖「開啟 FormIt」對話方塊

* 按一下 _**「3D 草圖」**_ 按鈕啟動 FormIt。選擇將所有可見物件或只將選取的物件帶入 FormIt。
  * 如果 Revit 模型很大，且設計只參考 Revit 模型的一部分，則選擇只帶入選取的物件會很有幫助。
  * 此外，在 Revit 中使用「剖面框」隔離 Revit 模型中要帶入 FormIt 的內部區域。

![](../.gitbook/assets/revit-objects.png)

#### 3D 草圖和樓層

* 如果選取「所有可見的 Revit 物件」，則任何可見的 Revit 樓層都將成為 FormIt 樓層，可以[在「樓層」選項板中檢視](../tool-library/levels-and-area.md#creating-and-configuring-levels)或[透過「性質」選項板套用至物件](../tool-library/levels-and-area.md#applying-levels)。
* 如果選取「選取的 Revit 物件」，則只有選取的樓層會成為 FormIt 樓層。

#### FormIt 啟動後

* FormIt 啟動時，內含的 Revit 幾何圖形將顯示在 FormIt 中自己的圖層上。此圖層稱為「Revit 環境」，預設會鎖住。
* 透過加入形狀或套用材料、內容等，在 FormIt 中建立模型。
* 從[「樓層」選項板](../tool-library/levels-and-area.md#applying-levels)對 FormIt 物件套用樓層。

### 傳送至 Revit 工作流程

* 使用「圖層」選項板中的 _**「傳送至 Revit」欄**_，納入或排除要傳送至 Revit 的幾何圖形。

![「圖層」選項板中的「傳送至 Revit」欄](../.gitbook/assets/revit-send-to-formit-layers-column.png)

* 您可以使用 _**Revit 預覽模式**_ 選擇預覽要傳送至 Revit 的幾何圖形。使用「圖層」選項板底部或「傳送至 Revit」按鈕旁邊的勾選方塊，啟用「Revit 預覽模式」。

![Revit 預覽模式](../.gitbook/assets/revit-preview-mode.png)

* 使用 FormIt 工具列上的 _**「傳送至 Revit」**_ 按鈕，將 FormIt 幾何圖形轉移到 Revit。
  * 這會提示您儲存檔案，將模型連結至 Revit。
    * **注意:** 我們建議將檔案儲存在本機，因為儲存至 Autodesk Docs 會中斷 FormIt 與 Revit 之間的連接。請改用 [Autodesk Desktop Connector](https://help.autodesk.com/view/CONNECT/ENU/) 將模型共用至雲端。
  * 建立連結後，只要想將變更轉移到 Revit，您可以重新儲存模型，或按一下「傳送至 Revit」按鈕。

![](<../.gitbook/assets/send-to-revit (2).png>)

* FormIt 圖層和材料將隨幾何圖形一起轉移到 Revit。
  * 您可以在 Revit 的「材料」對話方塊中依「FormIt」篩選清單，檢視 FormIt 材料。
  * 您可以在 Revit 視圖中透過「可見性圖形」的「匯入」頁籤，尋找 FormIt 圖層。在 Revit 中，這是一個展示不同 FormIt 設計選項的絕佳方式。
  * 若要將樓層傳送至 Revit，您需要透過「匯入 CAD」對話方塊匯入 AXM，並勾選底部的「匯入樓層」。

### 「以 FormIt 編輯」按鈕

* 在 Revit 中連結 FormIt 模型後，或使用 _**「3D 草圖」**_ 建立新的 FormIt 模型並儲存模型後，FormIt 元素會儲存至 Revit，稍後可再次編輯。
* 在 Revit 中選取 FormIt 元素，然後按一下「修改」頁籤中的 _**「以 FormIt 編輯」**_ 按鈕，在 FormIt 中啟動模型。

![](<../.gitbook/assets/20220809 FormIt-Revit Edit in FormIt.png>)

* 請參閱 FormIt 入門手冊，以取得 [FormIt-Revit 工作流程自學課程](../formit-primer/part-ii/2.8-advanced-revit-workflows.md)。

### **匯入和連結 FormIt 模型：在 FormIt 與 Revit 之間進行設計協同合作** <a href="#importing-formit-models-design-collaboration-between-formit-and-revit" id="importing-formit-models-design-collaboration-between-formit-and-revit"></a>

#### **建議的工作流程**

* 與同事共用 FormIt 模型，同事就可以將 FormIt 模型匯入他們的 Revit 專案供參考、視覺化和製作詳圖。
* 將 FormIt 模型直接匯入 Revit 設計選項，進一步探索和比較設計。由於 FormIt 圖層是透過匯入工作流程進入 Revit，因此 Revit 使用者可以控制哪些 FormIt 圖層在 Revit 設計選項中可見。
* 將 FormIt 模型匯入量體品類族群 (然後將其載入 Revit 專案)，以利用「樓層」、「量體樓板」和「依面」等工具，並對 FormIt 幾何圖形套用 Revit 系統族群。

![在 Revit 中匯入 CAD](<../.gitbook/assets/revit-workflows (2).png>)

* 將既有的 FormIt 模型連結至 Revit 以來回傳送變更、在重新載入期間更新幾何圖形和材料外觀，以及使用 FormIt 圖層控制幾何圖形的可見性。

![Revit 中的「連結 CAD」](../.gitbook/assets/revit-linking.png)

#### **使用工具**

* 指定要匯入至目前 Revit 專案的 FormIt 檔案。開啟 Revit 的 _**「匯入 CAD」**_ 工具。從可用檔案類型清單中選擇「.AXM」。瀏覽至 .AXM 檔案，然後按一下 _「開啟」_。
* FormIt 材料會轉移至 Revit，並顯示在 Revit 的「材料」對話方塊中。篩選清單找出 FormIt 材料。
  * 在 FormIt 中調整過比例或旋轉過的材料，在 Revit 中會保留正確的比例和旋轉。
* 將 FormIt 的樓層轉移至 Revit。
  * 在「匯入 CAD」對話方塊中按一下「匯入 FormIt 樓層」按鈕。
* FormIt 中彎曲/刻面幾何圖形的隱藏邊會進入 Revit，並帶有隱藏邊。
  * 已知問題：剪影邊在 Revit 的「隱藏線」視圖中也會顯示為隱藏。
* 為了可靠放置匯入的 FormIt 物件，會使用 FormIt 原點和 Revit 基準點。
* 匯入至 Revit 的 FormIt 幾何圖形將所有 FormIt 圖層一起匯入。
  * 您可以從「可見性圖形」>「匯入的品類」頁籤存取 FormIt 圖層。使用者在這裡將看到 FormIt 圖層的清單，可以使用勾選方塊將其開啟和關閉。
* Revit 的「依面」工具 (也稱為建築專家工具) 可從 FormIt 幾何圖形產生 Revit 元素
  * 這需要先將 FormIt 幾何圖形匯入至「量體族群」或「一般模型」族群，然後載入到專案環境。
* FormIt 幾何圖形會以一個匯入例證帶入。
  * 若要將 FormIt 模型以獨立的匯入內容來匯入，請在 FormIt 中隔離模型的零件，並將每個零件匯入為獨立的 .AXM 檔案。
* 在 [Revit 文件](https://help.autodesk.com/view/RVT/2023/ENU/?guid=GUID-A5F4B6C1-94CD-4433-A4E2-6B39B4BA8E6D)中進一步瞭解。

#### **使用 Revit 樓層和 FormIt 匯入**

* 將 FormIt 幾何圖形匯入 Revit 時，產生的物件可能因為 Revit 樓層而位於非預期的高程。
  * 在立面/平面視圖中，匯入的 FormIt 幾何圖形在 Revit 中將顯示在目前樓層。
  * 在 3D 視圖中，匯入的 FormIt 幾何圖形將預設為最低樓層，而這可能會低於地面。
  * 如果您的幾何圖形在 Revit 中以非預期的高度顯示，請選取匯入的物件，然後在 Revit 中將「基準樓層」調整到適當樓層。

#### **轉換 Revit 族群** <a href="#converting-revit-families" id="converting-revit-families"></a>

* 使用 Revit「增益集」頁籤上的 _**「將 RFA 轉換為 FormIt」**_ 按鈕將 Revit 族群轉換為 FormIt 內容，在 FormIt 環境中可以更有效率地重複使用既有的 Revit 內容。
* 指定 RFA 檔案的資料夾，這些檔案將轉換為 FormIt 檔案並輸出至您選擇的資料夾。
* 這是一個重複使用您既有所有 Revit 內容的絕佳方式。只需將內容轉換為 FormIt，並將 FormIt 的內容資源庫指向包含轉換檔案的資料夾即可。然後，從「內容資源庫」選項板中，按一下縮圖預覽並放置在模型中。

![](../.gitbook/assets/content-library.jpg)

#### **為 Revit 準備 SketchUp 檔案** <a href="#preparing-sketchup-files-for-revit" id="preparing-sketchup-files-for-revit"></a>

* 使用 FormIt for Windows 開啟或匯入 SketchUp 內容
* 選擇性使用[內建的診斷和修復工具修復幾何圖形](https://formit.autodesk.com/blog/post/using-formit-to-get-sketchup-data-into-revit)，讓幾何圖形更適合下游作業
* 儲存 FormIt 檔案，然後使用上述的 **「匯入 CAD」** 或 **「傳送至 Revit」** 工作流程匯入檔案

## FormIt 轉換器增益集的轉變 <a href="#revit-the-add-in" id="revit-the-add-in"></a>

FormIt + Revit 2022 和更高版本提供內建的 Synergy，無需額外下載。

因為這邊變更，增益集開始有所轉變。瞭解先前的工作流程如何改變。

### **匯入 AXM 檔案的新方法** <a href="#a-new-way-to-import-axm-files" id="a-new-way-to-import-axm-files"></a>

透過 Revit「插入」頁籤的「匯入 CAD」工具匯入 FormIt (.AXM) 檔案，取代增益集中的 _「將 FormIt 匯入至 RVT」_ 按鈕。

除了可加快高達 **30 倍**的速度，您會發現此工作流程通常也**更可靠**。除此之外，某些行為也改變了：

### **匯入時的幾何圖形**

* FormIt 幾何圖形會以一個匯入例證匯入。FormIt 品類不再成為 Revit 品類。我們正努力在未來的更新中解決此問題。
  * FormIt 品類目前只能用於能源分析。
* 您不必再將模型分成小塊 - 請嘗試一次傳送整個 FormIt 模型！
* 若要將 FormIt 模型以獨立的匯入內容匯入，請在 FormIt 中隔離模型的零件，並將每個零件匯入為獨立的 .AXM 檔案。

![](../.gitbook/assets/revit-old-new.png)

### **匯入時的材料**

* FormIt 材料轉移到 Revit 的狀況比以往更好，並可在「材料」對話方塊中排序。

![](../.gitbook/assets/revit-material-browser.png)

### **匯入後的工作流程**

* 對於涉及「量體樓板」和依面 (建築專家) 工具的工作流程，請先將 FormIt 模型匯入至 Revit 量體或「一般模型」族群，然後將其載入專案環境中。
* 此外，透過直接從 Revit 存取新的 3D 草圖繪製環境，可以讓工作更流暢！

### **Revit 族群的變更** <a href="#changes-to-revit-families" id="changes-to-revit-families"></a>

繼續使用 Revit「增益集」頁籤上的 _「將 RFA 轉換為 FormIt」_ 按鈕，在 FormIt 中使用 Revit 族群內容；此工作流程沒有改變。

我們努力提供更佳的體驗，因此已移除 _「重新載入族群」_ 按鈕。

![](../.gitbook/assets/revit-families-old-new.png)
